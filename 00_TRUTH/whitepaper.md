## 0. 한 줄 정의

**동선(DONGSEON)**은 동북아 부호 2세를 위한 **AI 왕도(王道) 교육** 스튜디오다.

> 우리는 AI를 가르치지 않는다. **"내 아들은 왕세자급"**이라는 아버지의 허세를 판다.

## 1. 현실 진단 (v2.1 추가)

### 1.1 냉정한 평가

| 항목 | 현재 점수 |
|------|----------|
| 실력 | 8/10 (25개 레포, 시스템 설계) |
| 권위 | 2/10 ("누구세요?") |
| 성공 사례 | 0/10 (없음) |
| 미디어 노출 | 0/10 (없음) |
| **총점** | **3.5/10** |

### 1.2 핵심 통찰

```
실력 8점 + 권위 2점 = 총점 3.5점

→ 현재 상태로 ₩3억 = 불가능
→ 해법: 권위에 맞는 가격으로 팔면서 권위를 쌓는다
→ 가격은 씰이 붙을 때마다 올라간다

"가격을 권위에 맞추고, 권위가 올라가면 가격도 올린다"
```

## 5. 문제 인식

### 5.1 AI 교육 시장의 착각

기존 AI 교육 시장은 이렇게 접근한다:

- "AI 사용법을 가르쳐드립니다"
- "프롬프트 엔지니어링 마스터"
- "ChatGPT 활용 100가지"

**틀렸다.**

### 5.2 진짜 수요

| 표면 수요 | 실제 수요 |
|-----------|-----------|
| AI 스킬 배우기 | **신분 과시** |
| 생산성 향상 | **"내 아들은 특별하다"** |
| 취업 경쟁력 | **계보 소속** |

**문제는 기능이 아니라 정체성이다.**

## 8. 네트워크 자본 모델

### 8.1 공식

```
V(n) = n × (n-1) / 2 × α

n = 문생 수
α = 1인당 연결 가치 (추정 1억원)

n=5  → 10억 관계 자산
n=10 → 45억 관계 자산
n=20 → 190억 관계 자산
```

### 8.2 차이점

| 기존 | 大提學 |
|------|--------|
| 매출 = 가치 | 매출 ≠ 가치 |
| 고객 = 소비자 | 고객 = **네트워크 노드** |
| 이탈 = 정상 | 이탈 = **신분 손실** |

## 9. 종교화 구조

### 9.1 師承 계보

```
대제학 (敎主) = 박씨
    │
    ├── 1대 문생 (長老)
    │       ├── 2대 문생 소개
    │       └── 네트워크 확장
    │
    ├── 師承 계보 = 정체성
    │       └── "나는 대제학 문하생"
    │
    └── 이탈 비용 = 극대화
            └── 떠나면 "계보 탈락" = 신분 손실
```

### 9.2 평생 고객 메커니즘

- **돈으로 못 사는 것** = 계보
- **계보** = 정체성
- **정체성** = 이탈 불가
- **이탈 불가** = 평생 고객
- **평생 고객** = 자녀도 입문
