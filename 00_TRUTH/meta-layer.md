# Meta Layer Whitepaper v2.0

> 大提學: 플레이어를 넘어서, 판의 문지기로

---

## 1. 문제 정의

AI를 처음 만난 대부분의 사람은 같은 지점에 도착한다.

- "AI로 돈 벌 수 있대"
- "자동화 시키면 편하대"
- "코딩 배워야 하나?"

**이건 틀린 생각이 아니다. 하지만 여기서 갈라진다.**

| 경로 | 결과 |
|------|------|
| 그냥 따라감 | 플레이어로 소모 |
| 거부함 | 순결한 가난 |
| 메타화 | 판을 파는 자리 |
| **大提學** | **판의 문지기** |

---

## 2. 기존 메타화 vs 大提學

### 2.1 기존 메타화 (v1.0)

```
[하위층] AI로 돈 벌고 싶은 사람
    ↓
[중간층] 실제로 굴리는 플레이어
    ↓
[상위층] 이 과정을 툴킷화해서 판매
```

**문제:** 여전히 "상품"을 판다. 고객은 1회성.

### 2.2 大提學 메타화 (v2.0)

```
[하위층] AI로 돈 벌고 싶은 사람
    ↓
[중간층] "내 아들을 왕세자로 만들고 싶은" 부호
    ↓
[상위층] 大提學 = 판의 문지기
    - 입문 허가권
    - 師承 계보
    - 네트워크 중심
```

**차이:** "상품"이 아니라 **"소속권"**을 판다. 고객은 평생.

---

## 3. 문지기 포지션

### 3.1 일반 교육 사업

```
콘텐츠 → 수강생 → 수료 → 끝
         ↑
      무한 경쟁
```

### 3.2 大提學 모델

```
                    ┌─────────────┐
                    │   大提學    │
                    │  (문지기)   │
                    └──────┬──────┘
                           │
           "입문 허가"      │      "계보 유지"
                    ┌──────┴──────┐
                    │             │
               문생 입문      문생 네트워크
                    │             │
                    └─────────────┘
                      평생 소속
```

**핵심:** 콘텐츠가 아니라 **"문을 열어주는 권한"**이 상품.

---

## 4. 왜 이게 강력한가

### 4.1 희소성

- 플레이어는 넘친다
- 강의하는 사람도 넘친다
- **"문지기"는 하나뿐**

### 4.2 이탈 불가

- 일반 교육: 수료 → 이탈 가능
- 大提學: **계보 = 정체성**, 이탈 = 자아 손실

### 4.3 네트워크 효과

| 문생 수 | 연결 수 | 관계 자산 |
|---------|---------|-----------|
| 5명 | 10개 | 10억 |
| 10명 | 45개 | 45억 |
| 20명 | 190개 | 190억 |

**문생이 늘수록 가치가 제곱으로 증가.**

### 4.4 세대 전승

- 1대 문생 → 자녀 입문
- 계보 = 가문의 자산
- **평생 + 다음 세대**

## 6. 실체의 역할

### 6.1 25개 레포 = 문지기의 자격 증명

```
"5주 만에 핸드폰으로 25개 만든 사람"
    ↓
"이 사람이 문을 열어주면, 나도 할 수 있다"
    ↓
"입문료 3억? 싸다"
```

### 6.2 스토리 = 허세의 근거

| 스토리 | 허세 변환 |
|--------|-----------|
| "핸드폰으로 개발" | "우리 아들도 어디서든 가능" |
| "말만 해서 코딩" | "왕세자가 신하에게 명령하듯" |
| "25개 레포 혼자서" | "제국을 세울 능력" |

---

## 7. 경쟁 불가 구조

### 7.1 왜 따라할 수 없나

| 요소 | 복제 난이도 |
|------|-------------|
| 25개 레포 | 시간 들이면 가능 |
| 핸드폰 개발 스토리 | **독학만 가능** |
| 師承 계보 | **시간 = 해자** |
| 문생 네트워크 | **선점자 우위** |

### 7.2 선점자 락인

```
Year 1: 첫 문생 5명 = 계보 시작
Year 3: "대제학 문하" = 브랜드
Year 5: 후발 주자가 와도 "짝퉁"
```
