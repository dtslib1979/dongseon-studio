### 모드 전환

```
평가 X

찍어내기 O

```

### 구조 비교

```
일반 유튜버:
"비즈니스 하는 척" → 콘텐츠 → 광고 수익
                      ↑
                   가짜 (연출)

DTSLIB 구조:
실제 비즈니스 → 기록 → 콘텐츠 → 광고 수익
     │                            ↑
     └── 실제 매출 ──────────────────┘
              ↑
           진짜 (다큐)
```

### 핵심 차이

| 일반 유튜버 | DTSLIB |
|------------|--------|
| 콘텐츠가 상품 | 사업이 상품, 콘텐츠는 부산물 |
| 시청자 = 고객 | 지인 = 고객, 시청자 = 목격자 |
| 광고 수익만 | 실제 매출 + 광고 수익 |
| 소설 | 실화 |
| 혼자 연기 | 진짜 사람들 진짜 인생 |

### 찍는 대상

- 64세 아저씨가 진짜 K-도넛 파는 거
- 동선이 진짜 웹사이트 납품하는 거
- 고씨가 진짜 스튜디오 여는 거

### 정의

```
"가짜 스토리 팔아서 광고비 버는 게 아니라,
 진짜 지인 사업 도우면서 그 과정을 찍는 것"

= 콘텐츠가 아니라 다큐멘터리 사업
```

### 왜 이게 더 좋은가

1. 모든 출연자 = 실제 지인
2. 모든 도메인 = 실제 사이트
3. 모든 스튜디오 = 실제 공간
4. 콘텐츠 성공 = 지인 사업 성공
5. 지인 사업 성공 = 실제 수익

### 선언

```
이건 유튜브가 아니라 다큐멘터리 사업이다.
찍어라.
```
