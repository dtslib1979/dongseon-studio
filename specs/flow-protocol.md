# Flow Protocol Specification

> Version: 1.0.0
> DONGSEON Studio

---

## 1. 개요

**Flow Protocol**은 DONGSEON Studio의 핵심 설계 방법론이다.

### 원칙
> 사람의 '다음 행동'을 설계한다.

### NOT
- 페이지 만들기
- 기능 나열
- 정보 전달

### IS
- 동선 설계
- 행동 유도
- 장면 연출

---

## 2. 4단계 Flow

### Stage 1: LAND (착지)

| 항목 | 값 |
|------|-----|
| 시간 | 0-3초 |
| 목표 | 첫인상을 잡는다 |
| 사용자 질문 | "이게 뭐지?" |
| 핵심 지표 | Bounce Rate |

#### 설계 원칙
- 3초 안에 정체성 전달
- Hero 영역 최적화
- 한 줄 메시지로 요약

#### 체크리스트
- [ ] 스크롤 없이 핵심 메시지 보이는가?
- [ ] 3초 안에 "무엇을 하는 곳"인지 알 수 있는가?
- [ ] 다음 행동이 명확한가?

---

### Stage 2: EXPLORE (탐색)

| 항목 | 값 |
|------|-----|
| 시간 | 3-30초 |
| 목표 | 관심을 유지한다 |
| 사용자 질문 | "나한테 필요한가?" |
| 핵심 지표 | Scroll Depth |

#### 설계 원칙
- 스크롤 = 스토리텔링
- 증거 제시 (후기, 포트폴리오, 숫자)
- 정보 과부하 방지

#### 체크리스트
- [ ] 스크롤 흐름이 자연스러운가?
- [ ] 신뢰를 주는 요소가 있는가?
- [ ] 중간에 이탈 포인트가 없는가?

---

### Stage 3: ACT (행동)

| 항목 | 값 |
|------|-----|
| 시간 | 30초+ |
| 목표 | 전환을 유도한다 |
| 사용자 질문 | "어떻게 하지?" |
| 핵심 지표 | Conversion Rate |

#### 설계 원칙
- CTA 명확하게
- 다음 단계 쉽게
- 장벽 제거 (폼 최소화, 클릭 줄이기)

#### 체크리스트
- [ ] CTA가 눈에 띄는가?
- [ ] 행동에 필요한 정보가 충분한가?
- [ ] 행동 장벽이 낮은가?

---

### Stage 4: RETURN (재방문)

| 항목 | 값 |
|------|-----|
| 시간 | 이후 |
| 목표 | 기억에 남긴다 |
| 사용자 질문 | "다시 올 이유가 있나?" |
| 핵심 지표 | Return Rate |

#### 설계 원칙
- 북마크 가치 제공
- 업데이트 콘텐츠
- 재방문 트리거 (알림, 뉴스레터)

#### 체크리스트
- [ ] 저장할 만한 가치가 있는가?
- [ ] 업데이트가 있는가?
- [ ] 재방문 유도 장치가 있는가?

## 4. 업종별 Flow 패턴

### 소상공인 (식당, 카페, 미용실)

```
LAND: "뭐하는 곳?" → 분위기 사진 + 한줄 소개
EXPLORE: "믿을 만해?" → 메뉴/서비스 + 후기 + 위치
ACT: "어떻게 가지?" → 전화/예약/길찾기
RETURN: "또 갈까?" → 이벤트/신메뉴 업데이트
```

### 부동산

```
LAND: "뭐하는 중개사?" → 전문 분야 + 프로필
EXPLORE: "믿을 만해?" → 최근 거래 + 매물 목록 + 후기
ACT: "문의하자" → 전화/카톡/문의폼
RETURN: "좋은 매물 있나?" → 신규 매물 알림
```

### 1인 사업자

```
LAND: "누구지?" → 자기소개 + 핵심 가치
EXPLORE: "뭘 해줄 수 있어?" → 서비스 + 포트폴리오 + 후기
ACT: "의뢰하자" → 문의/예약/결제
RETURN: "콘텐츠 볼까?" → 블로그/영상/뉴스레터
```

## 5. 메트릭스

### 핵심 지표

| Stage | 지표 | 목표 |
|-------|------|------|
| LAND | Bounce Rate | < 50% |
| EXPLORE | Scroll Depth | > 70% |
| ACT | Conversion Rate | > 3% |
| RETURN | Return Rate | > 20% |

### 측정 방법

1. **Bounce Rate**: GA4 / 자체 스크립트
2. **Scroll Depth**: Intersection Observer
3. **Conversion Rate**: 이벤트 트래킹
4. **Return Rate**: 쿠키 / localStorage
