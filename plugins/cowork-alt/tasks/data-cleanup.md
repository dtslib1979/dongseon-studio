# Task: 과거 데이터 청산 (일회성)

## 목적
PC 로컬에 쌓여있는 과거 엑셀/문서 파일 정리
→ 필요한 것만 추출 → GitHub 레포에 정형화 적재
→ 완료 후 이 태스크는 폐기

## 실행 조건
- PC 환경에서만 실행 (Cowork 또는 Claude Code PC)
- 대상 폴더 지정 필수

## 프로세스
1. 대상 폴더 스캔 → 파일 목록 생성
2. 파일별 내용 분석 → 보존/폐기 분류
3. 보존 대상 → resultset 스키마에 맞게 변환
4. GitHub 레포에 커밋
5. 로컬 원본은 박씨 판단에 따라 처리

## 분류 기준
- 보존: 내 모델 리절트셋, 내 시나리오 데이터, 내 설계 문서
- 폐기: 외부 리서치 사본, 백업 복사본, 남의 데이터 정리본
