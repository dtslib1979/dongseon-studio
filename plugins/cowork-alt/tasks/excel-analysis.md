# Task: 엑셀 시뮬레이션 콘솔

## 목적
엑셀을 장부가 아니라 시뮬레이션 실험대로 사용
내 모델 로직 → 파라미터 조정 → 결과 관찰 → 리절트셋 기록

## 실행 프로세스
1. scenario.json에서 시나리오 정의 읽기
2. 파라미터 세팅
3. 모델 실행 (Python/Claude Code)
4. 결과를 resultset.csv에 append
5. 필요시 추세선/회귀분석 추가 실행

## 금지 사항
- 과거 데이터 기반 선형 외삽 금지
- 피벗테이블만 돌리는 단순 집계 금지
- 박씨 가설 없이 "알아서 분석해줘" 금지

## 출력
- resultset.csv에 1행 추가 (시리즈 누적)
- 필요시 차트/리포트 생성 → 별도 파일
