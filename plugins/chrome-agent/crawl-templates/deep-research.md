결과는 판단 근거로만 사용하고 DB에 적재하지 않음

## 프로세스
1. 가설 수신 (박씨 → Claude/Gemini)
2. 딥리서치 실행
3. 결과 요약 → 판단 근거 제시
4. 박씨 판단: 맞으면 콘텐츠로 전환 / 아니면 방향 수정
5. 리서치 데이터 자체는 폐기 (저장 불필요)
