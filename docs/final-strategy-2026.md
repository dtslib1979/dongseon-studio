# Part 2: 네트워크 자본 모델

---

## 2.1 공식

```
V(n) = n × (n-1) / 2 × α

n = 문생 수
α = 1인당 연결 가치 (추정 1억원)
```

## 2.2 가치 증가 시뮬레이션

| 문생 수 | 연결 수 | 관계 자산 |
|---------|---------|-----------|
| 5명 | 10개 | 10억 |
| 10명 | 45개 | 45억 |
| 20명 | 190개 | 190억 |

**핵심:** 문생이 늘수록 가치가 **제곱**으로 증가.

## 2.3 기존 교육 vs 大提學

| 기존 교육 비즈니스 | 大提學 모델 |
|-------------------|-------------|
| 매출 = 고객 수 × 단가 | 자산 = 문생² × 연결 가치 × 시간 |
| 매출 = 가치 | 매출 ≠ 가치 (관계 자산이 진짜 가치) |
| 고객 = 소비자 | 고객 = **네트워크 노드** |
| 이탈 = 정상 | 이탈 = **신분 손실** |

# Part 3: 종교화 구조

---

## 3.1 師承 계보

```
대제학 (敎主) = 박씨
    │
    ├── 1대 문생 (長老)
    │       ├── 2대 문생 소개
    │       └── 네트워크 확장
    │
    ├── 師承 계보 = 정체성
    │       └── "나는 대제학 문하생"
    │
    └── 이탈 비용 = 극대화
            └── 떠나면 "계보 탈락" = 신분 손실
```

## 3.2 평생 고객 메커니즘

```
돈으로 못 사는 것 = 계보
    ↓
계보 = 정체성
    ↓
정체성 = 이탈 불가
    ↓
이탈 불가 = 평생 고객
    ↓
평생 고객 = 자녀도 입문 (세대 전승)
```

## 3.3 락인 비교

| 일반 교육 | 大提學 |
|-----------|--------|
| 수료 → 끝 | 입문 → **평생 師承** |
| 재구매 = 새 상품 필요 | 재구매 = **자녀 입문** |
| 이탈 비용 = 0 | 이탈 비용 = **신분 손실** |

# Part 5: 동북아 독점

---

## 5.1 경쟁 분석

| 시장 | 상태 | 문제 |
|------|------|------|
| 한국 | 코딩 학원 범람 | 기능 교육 (다른 시장) |
| 중국 | 정부 통제 | 외국 AI 접근 차단 |
| 일본 | 보수적 | 영어 장벽, 속도 느림 |
| **DONGSEON** | **유일** | **경쟁자 없음** |

## 5.2 독점 요인

1. **한자권 문화** + 師承 개념
2. **실체 시스템** (25개 레포 = 물증)
3. **私學 프레이밍** (학원 아닌 서원)
4. **핸드폰 개발** = 스토리

## 5.3 경쟁 불가 구조

| 요소 | 복제 난이도 |
|------|-------------|
| 25개 레포 | 시간 들이면 가능 |
| 핸드폰 개발 스토리 | **독학만 가능** |
| 師承 계보 | **시간 = 해자** |
| 문생 네트워크 | **선점자 우위** |

# Part 6: 25개 레포의 역할

---

## 6.1 레포 = 상품 ❌

레포는 판매 대상이 아니다.

## 6.2 레포 = 권위의 물증 ✅

| 차원 | 해석 |
|------|------|
| 기술 증명 | "코딩 없이 이 정도 만들 수 있다" |
| 신뢰 증명 | "말만 하는 놈 아님, 실체 있음" |
| 복제 증명 | "당신 아들도 이렇게 만들 수 있음" |
| 스토리 증명 | "이걸 혼자 한 사람한테 배운다" |

## 6.3 스토리 → 허세 변환

```
"5주 만에 핸드폰으로 25개 만든 사람"
    ↓
"이 사람이 문을 열어주면, 나도 할 수 있다"
    ↓
"입문료 3억? 싸다"
```

---

## 7.1 박씨 (大提學)

| 역할 | 내용 |
|------|------|
| 브랜드 | "대제학" 권위 유지 |

| 감독 | 품질 관리, 방향 설정 |

## 7.2 동선 (실행자)

| 역할 | 내용 |
|------|------|
| 실행 | 실제 과외/멘토링 진행 |
| 운영 | 일정, 고객 관리 |
| 현장 | 오프라인 접점 |
| 자립 | 3계정 후 독립 운영 |

## 7.3 자립 타임라인

```
Year 1: 1명 → 1억 (파일럿)
Year 2: 2명 → 6억 (동선 자립)
Year 3: 2명 → 6억 (박씨 = 감독만)

동선 자립 = 3계정 판매 완료
```

## 9.1 3 Type

| Type | 실체 | 역할 |
|------|------|------|
| **Type A** | 동선, 브랜치 5명 | "저걸 실제로 혼자서 굴리는 걸 봤다" |

- 문서에 Seed 쓰면 스팸 유입
