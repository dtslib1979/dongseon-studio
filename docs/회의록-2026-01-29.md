## 1. 동선의 욕망 구조 분석

### 동선이 말한 것

> "나는 돈을 벌고 싶다."

### 동선이 상상하는 AI 비즈니스

- 크롤링
- 쇼츠 자동 생성
- B급 콘텐츠 대량 생산
- 조회수 → 광고 수익
- "AI로 돈 벌었어요" 유튜브

### 공식화

```
AI = 생산기계
콘텐츠 = 원자재
나 = 운영자
목표 = 돈
```

### 특징

- 세계관 없음
- 장기성 없음
- 브랜드 없음
- 인간의 서사 없음
- "나"라는 캐릭터 없음

**진단: AI 초입자 90%가 빠지는 전형적인 패턴**

## 2. 박씨가 제시한 대안 구조

### 핵심 프레임

```
AI = 돈 찍는 기계 (X)
AI = 사람을 확장하는 외골격 (O)

콘텐츠 = 쓰레기 생산물 (X)
콘텐츠 = 삶의 흔적을 구조화한 기록 (O)

유튜브 = 노가다 수익 채널 (X)
유튜브 = 라이프스타일이 흘러나오는 포트 (O)
```

### 제안 모델

```
사람 1명
 └─ 이미 살고 있는 삶
     └─ AI로 자동화
         └─ 그 과정을 콘텐츠화
             └─ 브랜드화
                 └─ 수익화
```

### 핵심 문장

> "네 라이프스타일 동선 안에서 자연스럽게 나오는 걸 찍어라.
> 그러면 네 자체가 브랜드가 된다."

## 3. 갭 분석: 욕망 vs 구조

### 같은 것

| 항목 | 동선 욕망 | 현재 구조 |
|------|----------|----------|
| AI 활용 | O | O |
| 자동화 | O | O |
| 수익화 | O | O |
| 콘텐츠 생산 | O | O |

### 다른 것

| 항목 | 동선 욕망 | 현재 구조 |
|------|----------|----------|
| 콘텐츠 품질 | 쓰레기라도 OK | 실제 삶의 기록 |
| 시간 축 | 단기 (빠른 현금화) | 장기 (브랜드 축적) |
| 정체성 | 없음 | 있음 (동선 = IT 스튜디오) |
| 대상 | 불특정 다수 | 실제 고객 |
| 세계관 | 없음 | 있음 |

### 핵심 충돌

```
동선: "AI 돌려서 쇼츠 찍어서 광고비 받자"
구조: "실제 고객한테 납품하고 그 과정을 찍자"
```

## 4. 솔루션: 3-Lane 구조

### 결정 사항

**투트랙이 아니라 3-Lane으로 간다.**

```
DONGSEON 3-Lane Studio

Lane 1: 스튜디오 납품
        └─ 웹사이트, AI 연동, 패키지
        └─ 진짜 가치, 장기 브랜드
        └─ 고객: 소상공인, 부동산, 1인사업자
        └─ 수익: 납품비

Lane 2: 내 채널 실험
        └─ 쇼츠, 자동화, 크롤링 테스트
        └─ 내 데이터, 내 경험치
        └─ 수익: 광고, 조회수

Lane 3: 바이럴 용역
        └─ "우리가 하는 거 그대로 해드립니다"
        └─ 공개 운영 (투명하게)
        └─ 실제 바이럴 마케팅 업체처럼
        └─ 수익: 용역비
```

### 시너지 구조

```
Lane 2 실험 → Lane 3 상품화 → Lane 1 신뢰도 상승
     ↑                              │
     └──────────────────────────────┘
```

- Lane 2에서 삽질한 거 → Lane 3에서 팔고
- Lane 3에서 번 돈 → Lane 1 투자
- Lane 1 사례 → Lane 2, 3 콘텐츠

**3개가 서로 먹여살리는 구조**

## 5. 브릿지 전략

### 동선을 설득하는 방법

1. **첫 납품 1건 성사** (이번 주)
   - 동선이 "이게 되네" 경험
   - 50만원이든 100만원이든 실제 입금

2. **쇼츠 실험 허용** (사이드)
   - 하고 싶은 거 막으면 이탈함
   - Lane 2로 공식화

3. **쇼츠 기술을 납품 상품으로 전환**
   - "고객 가게 홍보 쇼츠" 패키지 추가
   - Lane 3로 공식화

4. **철학 말고 숫자로 말해라**
   - "이거 하면 월 200 가능" 식으로
   - 브랜드 얘기는 돈 들어온 다음에

## 6. 동선의 현재 위치

### 가진 것

- AI 사용 가능
- 툴 이해
- 웹 제작 가능
- 자동화 어느 정도 이해

### 없는 것

- 자기 삶을 '콘텐츠'로 보는 감각
- 자기 자신을 '브랜드'로 상상
- 장기 시간 축 (5년, 10년)
- 세계관

### 진단

> "1단계 인간"
> - AI를 '도구'로만 보는 단계
> - 돈을 '결과'로만 보는 단계
> - 삶을 '콘텐츠'로 인식하지 못하는 단계

**나쁜 게 아니라 아직 전환이 안 된 것**
