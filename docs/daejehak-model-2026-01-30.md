### 2.6 종교화 구조

```
대제학 (敎主)
    │
    ├── 1대 문생 (長老)
    │       ├── 2대 문생 소개
    │       └── 네트워크 확장
    │
    ├── 師承 계보 = 정체성
    │       └── "나는 대제학 문하생"
    │
    └── 이탈 비용 = 극대화
            └── 떠나면 "계보 탈락" = 신분 손실
```

**종교화의 핵심:**
- 돈으로 못 사는 것 = 계보
- 계보 = 정체성
- 정체성 = 이탈 불가
- 이탈 불가 = **평생 고객**

### 2.7 네트워크 효과 함수

```
V(n) = n × (n-1) / 2 × α

n = 문생 수
α = 1인당 연결 가치 (추정 1억원)

n=5  → V = 10 × 1억 = 10억
n=10 → V = 45 × 1억 = 45억
n=20 → V = 190 × 1억 = 190억
```

**문생 20명 = 네트워크 가치 190억**
(매출이 아닌 "관계 자산")

### 3.4 네트워크 자본 누락

#### 분석 보고서의 한계

```
매출 = 고객 수 × 단가
```

이건 **1회성 거래 모델**의 공식이다.

#### 대제학 모델의 진짜 공식

```
자산 = 문생 수² × 연결 가치 × 시간

n=5  → 10개 연결 × 1억 = 10억 관계 자산
n=10 → 45개 연결 × 1억 = 45억 관계 자산
n=20 → 190개 연결 × 1억 = 190억 관계 자산
```

**매출 ≠ 가치**

- Year 1 매출: 1억
- Year 1 네트워크 가치: 이미 측정 불가 (문생 1명 = 동북아 부호 2세 1명)

#### 종교화 락인 효과

| 일반 교육 | 大提學 |
|-----------|--------|
| 수료 → 끝 | 입문 → **평생 師承** |
| 재구매 = 새 상품 필요 | 재구매 = **자녀 입문** |
| 이탈 비용 = 0 | 이탈 비용 = **신분 손실** |

```
"대제학 문하생"을 그만둔다 = "나는 그 계보에서 탈락했다"를 인정
→ 이탈 불가
→ 평생 고객
→ 자녀도 입문
→ 세대 간 전승
```
