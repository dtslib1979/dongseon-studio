## Part 1: 핵심 컨셉 — 왕세자 놀이

### 1.1 워크플로우 발견

DONGSEON의 실제 워크플로우:
```
STT (음성 인식)
    ↓
Claude Code에 말로 명령
    ↓
"야, 이거 만들어" (욕해도 됨)
    ↓
AI가 코드 작성
    ↓
결과물 검수
    ↓
배포
```

**이것이 조선시대 왕세자가 내시에게 명령하는 것과 같다.**

### 1.2 패러다임 전환

| 평민의 방식 | 귀족의 방식 |
|------------|------------|
| 키보드로 타이핑 | 음성으로 명령 |
| 문법 오류 직접 수정 | AI가 알아서 처리 |
| Stack Overflow 검색 | 결과물만 검수 |
| AI가 시키는 대로 따라함 | AI가 나를 따름 |

### 1.3 마케팅 언어

```
❌ "AI 활용법을 배웁니다"
⭕ "AI를 부리는 법을 익힙니다"

❌ "프롬프트 엔지니어링"
⭕ "디지털 통치술"

❌ "학습"
⭕ "명령", "통치"
```

## Part 3: 6개월 커리큘럼

### Month 1: Foundation (기반 구축)
- GitHub 제국의 기초
- 25개 레포 = 25개 사업부 구조
- 0원 인프라 위에 디지털 영토 구축
- **결과물:** 본인 GitHub 제국 세팅

### Month 2: Voice Command (음성 통치)
- STT + Claude Code 조합
- 음성만으로 개발하는 워크플로우
- "야, 이거 만들어" 패턴
- **결과물:** 음성 명령으로 웹페이지 제작

### Month 3: Content Factory (콘텐츠 공장)
- 편집 0분 워크플로우
- PWA 칠판 + 화면 녹화 = 완성물
- 매일 콘텐츠 5개+ 생산 시스템
- **결과물:** 유튜브 채널 런칭

### Month 4: Automation (자동화 제국)
- 클로드앤크롬 GUI 자동화
- 반복 업무 0시간화
- AI가 브라우저 조작
- **결과물:** 일상 업무 자동화 3종

### Month 5: Data Sovereignty (데이터 주권)
- 박씨 캡처 APK로 AI 대화 수집
- RAG/파인튜닝 파이프라인
- 나만의 AI 준비
- **결과물:** 개인 대화 로그 아카이브

### Month 6: Empire (제국 완성)
- 모든 시스템 통합
- 브랜치 구조로 확장
- 독립 운영 가능 상태
- **결과물:** 완전한 디지털 제국
