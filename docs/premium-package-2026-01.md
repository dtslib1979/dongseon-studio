## Part 4: 네트워크 자본 모델

### 4.1 공식

```
V(n) = n × (n-1) / 2 × α

n = 문생 수
α = 1인당 연결 가치 (추정 1억원)
```

### 4.2 가치 증가

| 문생 수 | 연결 수 | 관계 자산 |
|---------|---------|-----------|
| 5명 | 10개 | 10억 |
| 10명 | 45개 | 45억 |
| 20명 | 190개 | 190억 |

**핵심:** 문생이 늘수록 가치가 **제곱**으로 증가.

### 4.3 기존 vs 大提學

| 기존 교육 | 大提學 |
|-----------|--------|
| 매출 = 가치 | 매출 ≠ 가치 |
| 고객 = 소비자 | 고객 = **네트워크 노드** |
| 이탈 = 정상 | 이탈 = **신분 손실** |

## Part 5: 종교화 구조

### 5.1 師承 계보

```
대제학 (敎主) = 박씨
    │
    ├── 1대 문생 (長老)
    │       ├── 2대 문생 소개
    │       └── 네트워크 확장
    │
    ├── 師承 계보 = 정체성
    │       └── "나는 대제학 문하생"
    │
    └── 이탈 비용 = 극대화
            └── 떠나면 "계보 탈락" = 신분 손실
```

### 5.2 평생 고객 메커니즘

```
돈으로 못 사는 것 = 계보
    ↓
계보 = 정체성
    ↓
정체성 = 이탈 불가
    ↓
이탈 불가 = 평생 고객
    ↓
평생 고객 = 자녀도 입문
```
