## Part 4: 동선 파일럿 (Phase 0)

### 4.1 역할 전환

| 기존 | 수정 |
|------|------|
| 동선 = 실행자 | **동선 = 첫 번째 고객 (파일럿)** |
| 첫 고객 = 외부에서 찾음 | **첫 고객 = 내부에서 만듦** |
| 성공 사례 = 미래 | **성공 사례 = 출시 전 완성** |

### 4.2 파일럿 구조

```
박씨 (大提學)
    │
    │ 사사 (전 노하우 전수)
    ↓
동선 (가상: 중국인 2세)
    │
    ├── GitHub 레포 생태계 구축
    ├── YouTube 채널 수익화
    ├── 방송국 운영
    ├── 브랜치 지인 IP화
    └── 실적 축적

    ↓ [6개월]

증빙 완성 → 판매 시작
```

### 4.3 파일럿 = 마케팅

```
"첫 고객을 찾지 마라. 첫 고객을 만들어라."

동선의 성공 = 살아있는 증빙
동선의 실적 = 판매 자료
동선의 과정 = 다큐멘터리 콘텐츠
```

## Part 10: 핵심 통찰 3가지

### 1. 실력 ≠ 판매

> *"실력 8점, 권위 2점이면 총점 3.5점. 실력만으로는 안 팔린다."*

### 2. 첫 고객 = 내부에서

> *"첫 고객을 찾지 마라. 만들어라. 동선이 살아있는 증빙이 된다."*

### 3. 씰 = 권위의 수학

> *"국가 + 종교 = 반박 불가. 3억이 싸 보이는 구조."*

## Part 11: 핵심 문구

### 한 줄 결론

> **"₩3억은 최종 목표. 지금은 권위에 맞는 가격으로 팔면서 권위를 쌓는다."**

### 판매 정의

> **"AI를 팔지 않는다. '내 아들은 왕세자급'이라는 아버지의 허세를 판다."**

### 권위 공식

> **"씰이 붙을 때마다 가격이 올라간다. 국가 + 종교 = 반박 불가."**

### 파일럿 원칙

> **"첫 고객을 찾지 마라. 만들어라. 동선이 살아있는 증빙이 된다."**
