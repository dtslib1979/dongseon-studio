## 1. 왜 외부 고객을 먼저 찾지 않는가

### 잘못된 접근

```
❌ 외부 고객 먼저 찾기
   └── "저 잘해요" (말만)
   └── 신뢰 0
   └── 가격 협상 당함
   └── 을 포지션
   └── 외주업체 취급
```

### 올바른 접근

```
✅ 내부 브랜치 먼저 성공시키기
   └── "저 했어요" (증거)
   └── 신뢰 100
   └── 가격 우리가 정함
   └── 갑 포지션
   └── 분양 사업자
```

### 비교표

| 항목 | 외부 고객 먼저 | 내부 브랜치 먼저 |
|------|---------------|-----------------|
| 신뢰도 | 말뿐 | 실적 증명 |
| 포지션 | 을 | 갑 |
| 가격 결정권 | 고객 | 우리 |
| 비즈니스 모델 | 외주 | 분양 |
| 스케일 | 1:1 노동 | 1:N 복제 |

## 4. AI vs Beyond AI 비교

### AI가 할 수 있는 것 (Phase 1)

```
• 코드 생성
• 콘텐츠 생성
• 자동화
• 스케일
• 정보 전달
• 번역/요약
• 이미지/영상 생성
```

### AI가 못하는 것 (Phase 3)

```
• 실제 공간 운영
• 실제 사람과 춤추기
• 물리적 만남
• 대구 지역 커뮤니티
• 경험 전달
• 냄새, 촉감, 분위기
• "거기 가봤어?"
• 신뢰 관계
• 로컬 네트워크
```

## 5. 경쟁자 진입 장벽 분석

### 경쟁자가 따라올 수 있는 구간

| 단계 | 경쟁자 시도 | 가능 여부 |
|------|------------|----------|
| 1 | "나도 AI 써야지" | ✅ 가능 |
| 2 | "나도 콘텐츠 만들어야지" | ✅ 가능 |
| 3 | "나도 자동화해야지" | ✅ 가능 |

### 경쟁자가 막히는 구간

| 단계 | 경쟁자 시도 | 장벽 |
|------|------------|------|
| 4 | "나도 스튜디오 차려야지" | 💰 자본 필요 |
| 5 | "나도 공간 확보해야지" | 💰 투자 필요 |
| 6 | "나도 커뮤니티 만들어야지" | ⏰ 시간 필요 |
| 7 | "나도 대구에서 해야지" | 📍 이미 선점 |

### 결론

```
AI까지 ─────────── 진입장벽 없음 (경쟁 구간)
물성화부터 ──────── 진입장벽 시작 (자본+시간)
Beyond AI ──────── 독점 (경쟁 불가능)
```
