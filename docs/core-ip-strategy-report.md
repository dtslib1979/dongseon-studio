## 1. AI 동영상 자동화 솔루션

### 정의

```
"찍으면 끝. 편집 0분."

핵심 개념:
├── 촬영 = 완성물 (후편집 없음)
├── PWA 칠판 + 화면 녹화 = 방송
├── MBC 선거방송급 인터랙티브 그래픽
├── 1인 + Galaxy 1대로 구현
└── 매일 업로드 가능한 지속성
```

### 구성 요소

| 요소 | 설명 | 구현체 |
|------|------|--------|
| **칠판 시스템** | 방송용 인터랙티브 배경 | OrbitPrompt PWA |
| **화면 녹화** | Galaxy 내장 녹화 기능 | 별도 장비 불필요 |
| **오버레이** | 실시간 판서 + 엑시스 | 터치 드로잉 |
| **에셋 파이프라인** | 이미지/오디오 자동 연동 | parksy-image, parksy-audio |
| **업로드** | 녹화 직후 바로 게시 | YouTube 쇼츠 |

### 워크플로우


## 2. 클로드앤크롬 GUI 자동화

### 정의

```
"브라우저가 곧 자동화 플랫폼"

핵심 개념:
├── Chrome + Claude = GUI 자동화
├── CDP (Chrome DevTools Protocol) 활용
├── 시각적 자동화 (픽셀 기반 아님)
├── 웹 앱 어디든 자동화 가능
└── RPA 대체 + 확장
```

### 구성 요소

| 요소 | 설명 | 기술 |
|------|------|------|
| **Claude** | 자연어 → 코드 변환 | Claude Code CLI |
| **Chrome** | 실행 환경 | Headless/Headful |
| **CDP** | 브라우저 제어 프로토콜 | playwright-bot |
| **스크린샷** | 상태 확인 | 자동 캡처 |
| **시나리오** | 반복 작업 정의 | JSON/Script |

## 3. GitHub 나처럼 쓰는 활용법

### 정의

```
"GitHub = 무료 서버 + 무료 호스팅 + 무료 CI/CD"

핵심 개념:
├── 서버비 0원으로 회사 전체 인프라 운영
├── GitHub Pages = 무료 웹 호스팅
├── GitHub Actions = 무료 자동화
├── 25개 레포 = 25개 사업부
└── 개씹사기 자산
```

### 활용 범위

| GitHub 기능 | 일반 용도 | DTSLIB 용도 |
|------------|----------|-------------|
| **Repos** | 코드 저장 | 사업부별 자산 관리 |
| **Pages** | 문서 호스팅 | 프로덕션 웹사이트 |
| **Actions** | CI/CD | 콘텐츠 자동화 파이프라인 |
| **Issues** | 버그 트래킹 | 업무 관리 |
| **Projects** | 프로젝트 관리 | 전사 로드맵 |
| **Wiki** | 문서화 | 내부 지식베이스 |
