# DONGSEON Studio 정관 v2.1

> 大提學 비즈니스 모델 + 프로덕트 피라미드
> 최종 수정: 2026-01-30

---

## 0. 한 줄 정의

**동선(DONGSEON)**은 동북아 부호 2세를 위한 **AI 왕도(王道) 교육** 스튜디오다.

> 우리는 AI를 가르치지 않는다. **"내 아들은 왕세자급"**이라는 아버지의 허세를 판다.

---

## 1. 현실 진단 (v2.1 추가)

### 1.1 냉정한 평가

| 항목 | 현재 점수 |
|------|----------|
| 실력 | 8/10 (25개 레포, 시스템 설계) |
| 권위 | 2/10 ("누구세요?") |
| 성공 사례 | 0/10 (없음) |
| 미디어 노출 | 0/10 (없음) |
| **총점** | **3.5/10** |

### 1.2 핵심 통찰

```
실력 8점 + 권위 2점 = 총점 3.5점

→ 현재 상태로 ₩3억 = 불가능
→ 해법: 권위에 맞는 가격으로 팔면서 권위를 쌓는다
→ 가격은 씰이 붙을 때마다 올라간다
```

## 4. 동선 파일럿 (Phase 0)

### 4.1 역할 전환

| 기존 | 수정 |
|------|------|
| 동선 = 실행자 | **동선 = 첫 번째 고객 (파일럿)** |
| 첫 고객 = 외부 | **첫 고객 = 내부 (동선)** |
| 증빙 = 나중에 | **증빙 = 출시 전 완성** |

### 4.2 파일럿 구조

```
박씨 (大提學)
    │
    │ 사사 (전 노하우 전수)
    ↓
동선 (가상: 중국인 2세)
    │
    ├── GitHub 레포 생태계 구축
    ├── YouTube 채널 수익화
    ├── 방송국 운영
    ├── 브랜치 지인 IP화
    └── 실적 축적

    ↓ [6개월]

증빙 완성 → Tier 1 판매 시작
```

### 4.3 파일럿 성공 = 마케팅

```
"첫 고객을 찾지 마라. 첫 고객을 만들어라."

동선의 성공 = 살아있는 증빙
동선의 실적 = 판매 자료
동선의 과정 = 다큐멘터리 콘텐츠
```

## 6. 네트워크 자본 모델

### 6.1 공식

```
V(n) = n × (n-1) / 2 × α

n = 문생 수
α = 1인당 연결 가치 (추정 1억원)

n=5  → 10억 관계 자산
n=10 → 45억 관계 자산
n=20 → 190억 관계 자산
```

### 6.2 종교화 구조

```
대제학 (敎主) = 박씨
    │
    ├── 1대 문생 → 2대 소개
    │
    ├── 師承 계보 = 정체성
    │   └── "나는 대제학 문하생"
    │
    └── 이탈 비용 = 극대화
        └── 떠나면 "계보 탈락" = 신분 손실
```
