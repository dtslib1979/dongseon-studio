## 0. 한 줄 정의

**동선(DONGSEON)**은 동북아 부호 2세를 위한 **AI 왕도(王道) 교육** 스튜디오다.

> 우리는 AI를 가르치지 않는다. **"내 아들은 왕세자급"**이라는 아버지의 허세를 판다.

## 2. 네트워크 자본 모델

### 2.1 공식

```
V(n) = n × (n-1) / 2 × α

n = 문생 수
α = 1인당 연결 가치 (추정 1억원)

n=5  → 10억 관계 자산
n=10 → 45억 관계 자산
n=20 → 190억 관계 자산
```

### 2.2 종교화 구조

```
대제학 (敎主) = 박씨
    │
    ├── 1대 문생 → 2대 소개
    │
    ├── 師承 계보 = 정체성
    │   └── "나는 대제학 문하생"
    │
    └── 이탈 비용 = 극대화
        └── 떠나면 "계보 탈락" = 신분 손실
```

### 2.3 평생 고객 메커니즘

| 일반 교육 | 大提學 |
|-----------|--------|
| 수료 → 끝 | 입문 → **평생 師承** |
| 재구매 = 새 상품 필요 | 재구매 = **자녀 입문** |
| 이탈 비용 = 0 | 이탈 비용 = **신분 손실** |
